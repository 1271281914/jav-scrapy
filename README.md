#jav-scrapy

jav-scrapy，一个爬取 AV 种子的小爬虫。

## Prequisites

- Node.js 4.2.1+

## Installation

```bash
git clone https://git.coding.net/raawaa/jav-scrapy.git
cd jav-scrapy
npm install # 安装npm包依赖
```

## Usage

```bash
node index.js --harmony # 获取磁力链接并打印至标准输出，可自己使用重定向保存到文本文件中
```
